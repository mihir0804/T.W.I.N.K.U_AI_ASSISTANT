## ✨ Features

- 🧠 Smart App Detection (Alias + Partial Match)
- ⚡ Instant App Launch
- 🖥️ Windows Support (Win32 + UWP Apps)
- 🔄 Multi Method Execution System:
  - CMD (`start`)
  - PowerShell (`Start-Process`)
  - Start Menu Scan
- 🚀 Fast & Lightweight
- 💡 Beginner Friendly Code
